# i3 config (unused)

X11 window manager config. Source: `i3.nix`.

## Base

- **Modifier:** `Mod4` (Super)
- **Terminal:** `kitty`
- **Font:** DejaVu Sans Mono 10pt (window titles and i3bar)
- **Bar:** `i3bar` + `i3status`

## Startup

| Command | Purpose |
|---|---|
| `xss-lock --transfer-sleep-lock -- i3lock` | Auto-lock on sleep/suspend; transfers sleep lock so screen locks before system sleeps |
| `nitrogen --restore` | Restore wallpaper |
| `nm-applet` | Network tray applet |

`xsession.profileExtra` starts `gnome-keyring-daemon` with gpg, pkcs11, secrets, ssh components.

## Keybindings

| Key | Action |
|---|---|
| `Mod4+m` | Move workspace to output left |
| `Mod4+Shift+m` | Move workspace to output right |
| `Mod4+b` | Reset wallpaper (`nitrogen --restore`) |
| `Mod1+Space` | `rofi -show drun` |
| `Mod1+Tab` | `rofi -modi combi -combi-modi window` (window switcher) |
| `Mod4+Shift+x` | Enter system mode |

## System mode (`Mod4+Shift+x`)

`System (l) lock, (e) logout, (s) suspend, (h) hibernate, (r) reboot, (d) shutdown`

| Key | Action |
|---|---|
| `l` | `i3lock` |
| `e` | i3-nagbar warning → `i3-msg exit` |
| `s` | `i3lock` then `systemctl suspend` |
| `h` | `i3lock` then `systemctl hibernate` |
| `r` | i3-nagbar warning → `systemctl reboot` |
| `d` | i3-nagbar warning → `systemctl poweroff` |
| `Return` / `Escape` | Back to default mode |

Logout/reboot/shutdown require nagbar confirmation ("All unsaved data will be lost."). Suspend/hibernate lock screen first without confirmation.
